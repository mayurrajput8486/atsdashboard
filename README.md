
# ATS Dashboard Project

A responsive ATS (Applicant Tracking System) Dashboard built using React.js and Vite.

## Features

- Login Page
- Dashboard Overview
- Job Listing Page
- Candidate Listing Page
- Candidate Details Page
- Responsive Design
- Reusable Components

## Tech Stack

- React.js
- Vite
- React Router DOM
- CSS

## Installation

```bash
npm install
npm run dev
```

## Project Structure

```bash
src/
 ├── components/
 ├── pages/
 ├── data/
 ├── assets/
 └── App.jsx
```

## Deployment

Deployed using Vercel.
